The What and the Why

Color has an undeniable power in web design. Everyone can recall a time when they were turned off by neon font on a black background - a far cry from legibility.

But color and design often take a backseat to the functionality that developers are so eager to achieve. Other times, it's a matter of not having the tools to determine specific color (Hex/RGB) values to implement when coding.

Thus came the inspiration for this project - a way to visualize and get values of various colors splashed onto a sample web layout.


Behind the Scenes

When you select a color with Angular's color picker, it passes its associated hex value to an external API called "The Color API". The API returns data pertaining to that color, and this application then uses that data to compose the color palette and sample UI.

Once you're happy with the colors you've selected, clicking "Get JSON Data" provides the option to download a JSON file with all associated data for future reference.
